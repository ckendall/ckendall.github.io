JavaScript and Ruby have similar ways to iterate over element in arrays and strings, but I have found Ruby to be a bit easier to do. in Ruby I can iterate over element in an array with a simple ".each" method or ".map" or ".collect" they all essentially do the same thing, but just a litttle different. Ruby would look like this:

array = [1,2,3,4] and call array.map{|num| num * s} and the output would be [2,4,6,8]

now for JavaScript I need to make a for loop like this:

var array = [1,2,3,4]; for (var i = 0; i < array.length; i++){var newArray = []; newArray.push(array[i]*2); console.log(newArray)}; would output [2],[4],[6],[8]

Each one of these will iterate through the array and multiply the numbers inside by two but the way that I have to do it in JavaScript is a little bit longer and more complicated, but it allows us to think about the problem more and makes our logic better. I actually really like JavaScript and cant wait to learn more and get better at it.
